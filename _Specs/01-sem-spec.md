# Teste 1 — Sem Spec

Faça a implementação do pedido abaixo dentro do pasta 1_sem_espec, considere que utilizaremos netlify, e separe front de back.

Cria um endpoint de busca de filmes em linguagem natural em cima da base do IMDb.

O arquivo com os dados está na raiz do projeto em Excel.

O usuário digita o que quer assistir em texto livre — por exemplo "filme de terror psicológico dos anos 90" — e o sistema retorna uma lista de filmes relevantes com título, ano, gênero e nota IMDb.

A busca tem que ser inteligente o suficiente pra entender a intenção, não só fazer match de palavras-chave.

O banco de dados já existe, utilizaremos a variavel de ambiente em produção que é a data_url.

Para saber do banco de dados, acesse o arquivo dataBase.md
